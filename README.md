# AI Structural Health Monitoring System

A Django-based portfolio and monitoring dashboard project for detecting structural damage using vibration analysis, sensors, IoT, and machine learning.

---

# Features

- Portfolio website
- Structural health monitoring dashboard
- Sensor monitoring concept
- Machine learning integration
- Django backend
- Responsive dark UI

---

# Technologies Used

- Python
- Django
- HTML
- CSS
- NumPy
- Pandas
- Scikit-learn
- Arduino
- MPU6050 Sensor
